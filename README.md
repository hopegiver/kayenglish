## 📖 각 챕터 구성

모든 챕터는 다음과 같은 구조로 되어 있습니다:

```
📌 핵심 개념
  └─ 기본 패턴 설명

📚 주요 단어
  └─ 필수 어휘 리스트

💬 주요 표현
  └─ 실전 표현 모음

🗣️ 회화 예제
  └─ 실제 대화 상황

✍️ 연습문제
  └─ 빈칸 채우기, 문장 완성

🎯 패턴 훈련
  ├─ Step 1: 기본 패턴 익히기
  ├─ Step 2: 표현 연습 (한글→영어)
  └─ Step 3: 실전 대화 연습

💡 학습 팁
  └─ 효과적인 암기법
```
